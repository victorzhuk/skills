# Writing Skills

Documentation and prose cleanup skills for concise, human text.

## Model-invoked

Model- or user-reachable via skill name and trigger phrasing.

- **[z-concise-human-docs](./z-concise-human-docs/SKILL.md)** — Write and edit documentation, comments, guides, runbooks, specs, and explanatory prose in a concise human style. Does not cover publishable prose or voice cleanup; see [[z-fuck-slop]].
- **[z-fuck-slop](./z-fuck-slop/SKILL.md)** — De-slop pass for any text: detects and erases the statistical fingerprints of AI writing (negative parallelism / "not X but Y", em-dash abuse, rule-of-three, false ranges, puffery vocabul…
- **[z-ru-tech-register](./z-ru-tech-register/SKILL.md)** — Russian tech prose in the native senior-engineer register — Latin-vs-Cyrillic term choice, serious-register structure, RU neuro-slop tells, and the bilingual RU/EN terminology glossary by theme cluster. Telegram post format lives in [[z-tg-post]].
- **[z-tg-post](./z-tg-post/SKILL.md)** — Write, rewrite, or review short-form posts for the author's Telegram tech blog in his voice — Telegram user-Markdown formatting (HTML for bot/API), post structure, the compressed anti-slop voice. Register and glossary live in [[z-ru-tech-register]].
