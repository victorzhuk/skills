# Writing Skills

Documentation and prose cleanup skills for concise, human text.

## Model-invoked

Model- or user-reachable via skill name and trigger phrasing.

- **[z-concise-human-docs](./z-concise-human-docs/SKILL.md)** — Write and edit documentation, comments, guides, runbooks, specs, and explanatory prose in a concise human style.
- **[z-fuck-slop](./z-fuck-slop/SKILL.md)** — De-slop pass for any text — detects and erases AI-writing fingerprints (negative parallelism, em-dash abuse, rule-of-three, false ranges, puffery, uniform cadence) and rewrites into its target…
- **[z-ru-tech-register](./z-ru-tech-register/SKILL.md)** — Russian tech prose in the native senior-engineer register — Latin-vs-Cyrillic term choice, serious-register structure, erasing tells that mark a text as machine-written.
- **[z-tg-post](./z-tg-post/SKILL.md)** — Write, rewrite, or review short-form posts for the author's Telegram tech blog in his voice.
