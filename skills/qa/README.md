# QA Skills

QA planning and autonomous API, browser, visual, performance, and failure-triage workflows.

## Model-invoked

Model- or user-reachable via skill name and trigger phrasing.

- **[z-qa-analyst](./z-qa-analyst/SKILL.md)** — Codebase research, API surface mapping, TODO/FIXME triage, and prioritized test-plan authoring with ripgrep.
- **[z-qa-api](./z-qa-api/SKILL.md)** — Autonomous API testing with hurl (declarative HTTP) and xh (fast calls).
- **[z-qa-browser](./z-qa-browser/SKILL.md)** — Autonomous browser testing with playwright-cli (token-efficient CLI) and Playwright MCP.
- **[z-qa-debugger](./z-qa-debugger/SKILL.md)** — Test execution without fail-fast, failure classification, flaky detection, and QA report authoring.
- **[z-qa-orchestrator](./z-qa-orchestrator/SKILL.md)** — QA orchestration: coordinates browser/API/visual/performance agents, test planning, parallel execution, and report synthesis.
- **[z-qa-performance](./z-qa-performance/SKILL.md)** — Performance and load testing with k6, oha, vegeta, and Lighthouse CLI.
- **[z-qa-spec-writer](./z-qa-spec-writer/SKILL.md)** — Test framework auto-detection and compilable test stub + Gherkin BDD spec generation.
- **[z-qa-visual](./z-qa-visual/SKILL.md)** — Visual regression testing, screenshot comparison, and automated accessibility test execution.
